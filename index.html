I've only used it one day so far, so I'm not here to write a review about its longevity. However, I can speak to how solidly it is made and how easy it is to operate. The controls are straightforward, and I was able to quickly set a couple of different heights with the memory buttons.

The motor is very quiet and not intrusive at all. I have migraines, so that part is important to me (mentioning it in case this applies to you too).
